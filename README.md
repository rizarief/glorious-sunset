# Installation
Go to **Themes** under **Revenge** in **Settings**, then tap the (+) button, finally paste this URL:

```
https://github.com/eeriemyxi/gruvbox-theme-revenge/raw/refs/heads/main/gruvbox-theme-revenge.json
```
